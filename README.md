# Sort Visualizer

https://mwronski12.github.io/sort-visualizer/

## To run:

- Make sure You have node installed
- `git clone https://github.com/MWronski12/sort-visualizer`
- `cd sort-visualizer`
- `npm install`
- `npm run dev`
- Open in browser

## Available sorts:

#### Quadratic time:

- Bubble Sort
- Insertion Sort
- Selection Sort

#### Linearithmic time:

- Shell Sort
- Merge Sort
- Quick Sort
